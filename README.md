# Jogo da Forca

![](https://imgur.com/ZxRKHg9.gif)

Este é um simples jogo de forca em console, onde o jogador escolhe uma categoria (frutas, animais ou países) e tenta adivinhar uma palavra secreta. O jogo termina quando o jogador acerta a palavra ou atinge o limite de erros.

## Como Jogar

1. **Escolha uma Categoria**: O jogador pode escolher entre três categorias:
   - Frutas
   - Animais
   - Países

2. **Adivinhe as Letras**: O jogador deve tentar adivinhar a palavra secreta, letra por letra. O jogador pode digitar uma única letra ou a palavra inteira. Cada tentativa errada aumenta o número de erros e aproxima o jogador do "enforcamento".

3. **Regras**:
   - O jogador tem no máximo 6 tentativas (erros).
   - Se o jogador acertar a palavra secreta, o jogo termina e ele vence.
   - Se o jogador atingir o limite de tentativas erradas, o jogo termina e ele perde.
